# 🐾 Золотая Лапка — сайт питомника британских золотых шиншилл

Одностраничный сайт заводчика на **Next.js 14** (App Router, TypeScript), без лишних зависимостей.

## 🚀 Запуск

```bash
npm install
npm run dev
```

Открой http://localhost:3000 — готово.

Продакшен-сборка:

```bash
npm run build
npm start
```

## ✏️ Где что менять

| Что | Где |
| --- | --- |
| Название, телефон, соцсети, город | `data/site.ts` → объект `site` |
| Котята (имена, окрасы, цены, статусы) | `data/site.ts` → массив `kittens` |
| Производители | `data/site.ts` → `parents` |
| Отзывы и достижения | `data/site.ts` → `reviews`, `achievements` |
| Вопросы-ответы | `data/site.ts` → `faq` |
| Цвета и стили | `app/globals.css` (CSS-переменные сверху) |

## 📸 Как добавить настоящие фото

1. Положи фотографии в `public/gallery/` (например, `public/gallery/kitten1.jpg`).
2. В `app/page.tsx` в секции «Галерея» замени эмодзи-плейсхолдеры на `<Image>` из `next/image`:

```tsx
import Image from "next/image";

<Image src="/gallery/kitten1.jpg" alt="Котёнок Оскар" width={400} height={340}
  style={{ objectFit: "cover", borderRadius: 26, width: "100%", height: "100%" }} />
```

Аналогично можно заменить эмодзи в карточках котят (`kitten-photo`) и у родителей.

## 📨 Подключение формы заявок

Форма в `components/ContactForm.tsx` сейчас работает локально (показывает «отправлено»).
Чтобы заявки реально приходили, подключи один из вариантов:

- **Formspree / Getform** — самый быстрый способ: замени `onSubmit` на `fetch` к их endpoint.
- **Telegram-бот** — создай route handler `app/api/lead/route.ts` и шли сообщение через Bot API.
- **Почта** — Resend / Nodemailer в том же route handler.

## 🌍 Как выложить сайт в интернет (бесплатно, без опыта)

Самый простой способ — **Vercel** (создатели Next.js, бесплатный тариф):

1. Зарегистрируйся на [github.com](https://github.com) и на [vercel.com](https://vercel.com) (можно войти через GitHub).
2. Создай на GitHub новый репозиторий и загрузи туда папку проекта
   (на странице репозитория: **Add file → Upload files** — можно просто перетащить все файлы).
3. На Vercel нажми **Add New → Project**, выбери свой репозиторий и нажми **Deploy**.
   Vercel сам поймёт, что это Next.js — ничего настраивать не нужно.
4. Через минуту сайт будет доступен по адресу вида `zolotaya-lapka.vercel.app`.
5. Свой домен (например, `zolotaya-lapka.ru`) можно купить у любого регистратора
   и привязать в настройках Vercel: **Settings → Domains**.

После любых правок просто загружай обновлённые файлы в GitHub — Vercel пересоберёт сайт автоматически.

## ✨ Анимация при скролле

Секции и карточки плавно «всплывают» при прокрутке (`components/ScrollReveal.tsx`).
Хочешь добавить эффект любому блоку — просто поставь ему атрибут `data-reveal`.
Если у посетителя включено «уменьшение движения» или выключен JS — контент виден сразу.
